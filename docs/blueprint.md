# **App Name**: GlamTutor

## Core Features:

- AI Makeup Analysis: Analyze uploaded face images to detect face shape, skin tone, current makeup, eye color, and lip shape, and generate personalized makeup routines using AI-driven decision logic. The AI will use tool to decide the ideal routine.
- AI Product Recognition: Process uploaded makeup product images to identify the brand, product name, and use-case. Return usage guidance, trusted articles, ingredients (if possible), and suggest better alternatives/dupes. The AI will use a tool to find similar products.
- AI Feedback Loop: Incorporate a dynamic loop system where users can request variations with prompts like “show another”, “simpler”, or “different vibe” to iterate on makeup looks. The AI will use a tool to incorporate the user feedback to make better decisions.
- Beginner Mode: Offer pre-curated beginner-friendly routines using simple product sets and detailed, tooltip-supported instructions. Embed YouTube tutorials for visual learners.
- AI Chatbot Assistant: Integrate a conversational assistant that:

Answers makeup-related questions.

Explains product usage and techniques.

Recommends looks or routines.

Links to trusted tutorials via YouTube Data API.
- Privacy-Focused Design: Implement guest mode where no data (images, chat, usage) is saved after the session ends. Optionally allow users to create accounts to save looks, routines, and history securely.
- Educational Explanations: Provide contextual explainers triggered by product type, technique, or routine step. These include ingredient info, brand background, technique definitions, and tutorial suggestions. AI tool for deciding what information is helpful to the user.

## Style Guidelines:

- Primary color: Deep magenta (#D90368) to convey sophistication and glamour.
- Background color: Soft, desaturated pink (#F5E6EA) for a calming and approachable feel.
- Accent color: Vivid violet (#6D597A) for interactive elements and highlights, offering a touch of mystery.
- Headline font: 'Playfair', a modern sans-serif, similar to Didot, with an elegant, fashionable, high-end feel.
- Body font: 'PT Sans' (sans-serif), which will be paired with Playfair.
- Use elegant, minimalist icons that reflect beauty and makeup themes.
- Maintain a clean and mobile-friendly layout that prioritizes image clarity and step-by-step guidance.
